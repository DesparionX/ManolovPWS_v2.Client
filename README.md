# ManolovPWS

Personal portfolio site for Manolov — blog posts, projects, and a CV, with a single-user (Owner-only) admin area for managing content. No public sign-up.

React + TypeScript client, backed by a separate .NET API (not in this repo).

## Stack

React · TypeScript · Vite · React Router · TanStack Query · React Hook Form · Zod · Tailwind CSS
